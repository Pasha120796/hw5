Селекторы бывают:

  • по названию элемента;
  • по id;
  • по классу;
  • по нескольким классам;
  • по вложенности.
по названию элемента - этот базовый селектор выбирает тип элементов, к которым будет применяться правило;
по id - этот базовый селектор выбирает элементы, основываясь на значении их id атрибута;
по классу - этот базовый селектор выбирает элементы, основываясь на значении их атрибута class;
по нескольким классам - '~' выбирает элементы, которые находятся на этом же уровне вложенности, с тем же родителем;
по вложенности - '>' в отличие от пробела выбирает те элементы, которые являются дочерними по отношению к указанному элементу.
